📘 Travel Journal

A modern React-based travel journal web application that allows users to document and showcase their travel experiences with images, locations, and descriptions.

📌 Features
🌍 Add and display travel locations
🖼️ Show images for each destination
📍 Direct link to location via Google Maps
📝 Store travel details (title, country, description)
⚡ Fast and responsive UI using Vite
🎨 Clean and simple user interface
🛠️ Tech Stack
Frontend: React.js
Build Tool: Vite
Styling: CSS
JavaScript: ES6+
